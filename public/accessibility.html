Accessibility Statement

← sherpacarta.org

SherpaCarta aims to meet WCAG 2.1 Level AA where practicable for a zero-tracking static site.

Features

  • Skip to content and section skip links
  • Keyboard navigation and ⌘K command palette
  • Font size, contrast, and reading mode tools
  • prefers-reduced-motion support
  • ARIA labels on navigation, modals, and forms
  • RTL layout for Arabic

Known limitations

  • Custom cursor disabled when reduced motion is on; system cursor preferred for accessibility
  • Charter modal density may challenge screen readers on mobile; focus trap improvements ongoing
  • Campaign signature totals are rate-limited self-reports, not identity-verified
  • Rights Dispatch email list is not live yet (waitlist via email only)
