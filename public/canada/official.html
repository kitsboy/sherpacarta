How to go live (Cam checklist)

  1. Create account on petitions.ourcommons.ca
  2. Paste the federal prayer above (no URLs)
  3. Add 5–10 supporters with emails (not yourself)
  4. Invite an MP to authorize online publication
  5. Clerk certifies → you receive e-XXXX
  6. Update data/campaign-canada.json → officialChannels.federal.status = "live", set ePetitionId + url → redeploy
