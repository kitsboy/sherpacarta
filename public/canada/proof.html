How verification works

  1. Petition hash — SHA-256 of the canonical campaign prayer text. Same text → same hash for everyone.
  2. Receipt hash — each campaign sign produces a local SHA-256 receipt (optional sync of hash + province + method only).
  3. Merkle root — combine this device’s receipts into one root; stamp via Satohash / OpenTimestamps on Bitcoin.
  4. Not Parliamentary proof — official e-petition signatures live only on ourcommons.ca after MP authorization.
